Revisión bucles for

  • Hemos hecho repeticiones con for
  • Lo usamos para:
    • Recorrer una secuencia
    • Repetir un número fijo de veces algo
  for var in secuencia:
      bloque

  for x in range(10):
      print x

Bucles no fijos

  • No siempre sabemos cuántas veces va a repetirse algo
  • Un profesor introduce notas ¿cuántas?
  • ¿Cuántas canciones hay en tu mp3?
  • Idea: repetir algo mientras se cumple una condición
  • 	while condición:
	    bloque
      
    	# lista de números positivos
	num = input("Introduzca número: ")
	while num >= 0:
	    print num
	    num = input("Introduzca número: ")

Patrones comunes: bucles interactivos

Modelo

  • El usuario controla el bucle en todo momento (se le pregunta cada vez si quiere repetir)
    poner más_datos a sí
    while más_datos es sí:
        tomar el siguiente dato
	procesar el dato
	preguntar si hay más datos

Ejemplo:

    suma = 0.0
    contador = 0
    mas_datos = 'si'
    while mas_datos == si:
        x = input("Introduzca un número >> ")
	suma = suma + x
	contador = contador + 1
	mas_datos = raw_input("¿Hay más números? (si/no)  ")
    print "La media es", suma/contador

Patrones comunes: centinela

  • El bucle continúa hasta que se alcanza un valor especial que indica el fin
      suma = 0.0
      contador = 0
      x = input("Introduzca un número positivo: ")
      while x >= 0:
          suma = suma + x
	  contador = contador + 1
      print "La media de los números es", suma / contador
