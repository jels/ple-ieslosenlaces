¿Qué es un objeto?

  • Una entidad que agrupa
    • estado
    • una funcionalidad
    relacionadas.
  • Estado del objeto: a través de variables llamadas atributos
  • funcionalidad: a través de funciones: métodos

¿Ventajas? ¡Todas!

  • Tenemos juntas información y funcionalidad
  • Podemos reutilizar
  • Podemos hacer copias de nuestros objetos
  • Ofrecemos una forma de usar el objeto y podemos cambiarlo internamente

Ejemplo

Héroes y aliens

  • El héroe tiene que eliminar al alien
  • Podemos crear héroes que actúen con aliens
  • Creamos aliens que respondan a las acciones de un héroe

Ejemplo (2)

Héroes y aliens

Definición

class Heroe(object):
    "Un jugador de la partida."""
    def ataca(self, enemigo):
        print "El jugador dispara al enemigo"
        enemigo.huir()

class Alien(object):
    "Un alien que invade el planeta"
    def huir(self):
        print "Me voy porque me atacan ..."

Uso

jugador = Heroe()
enemigo = Alien()
jugador.ataca(enemigo)
